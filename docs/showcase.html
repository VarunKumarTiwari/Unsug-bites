Component Showcase

The Unsung Bites design system. Tokens live in packages/ui/src/tokens/; primitives in packages/ui/src/primitives/; domain components in apps/mobile/components/. Everything is imported from @unsung/ui (or local feature paths).

Colors

Two-layer system: private palette (raw values) → public color (semantic roles). Components only consume the semantic roles. To re-theme, swap the role map.

Motion

Three named springs cover everything in the home feed today. Never inline { damping, stiffness, mass }.

Springs

TokenUseValues
spring.gentleSheets, modals, screen entries{ damping: 22, stiffness: 200, mass: 1 }
spring.snappyToggles, pills, list-item returns{ damping: 18, stiffness: 260, mass: 0.7 }
spring.bouncyPress-release on chips and cards{ damping: 9, stiffness: 260, mass: 0.55 }

Durations

TokenUseValue
duration.microPress compression80ms
duration.fastQuick fades, color shifts, dismissals150ms
duration.baseEntry fades, slide-ins300ms
duration.slowHero / section staggered entries400ms
duration.ambientShimmer loops, ambient parallax650ms

Easings

TokenUse
easing.outEntries, fades-in, settles
easing.inExits, compressions, dismissals
import { spring, duration, easing } from '@unsung/ui';

// Tap-down then settle
onPressIn:  scale.value = withTiming(0.91, { duration: duration.micro });
onPressOut: scale.value = withSpring(1, spring.bouncy);

// Sheet entry
<Animated.View entering={SlideInUp.duration(280).springify().damping(spring.gentle.damping).stiffness(spring.gentle.stiffness)} />

HiddenGemBadge

Renders only when score >= 0.7. The signature "Hidden Gem" pill that appears on featured restaurant cards.
